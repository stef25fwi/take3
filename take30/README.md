# Take30

Application Flutter structurée pour un MVP complet.

## Dossiers
- lib/screens : écrans de l’application
- lib/router : navigation
- lib/services : données simulées et services
- lib/widgets : composants partagés
- lib/theme : thème visuel

## CI/CD
- build Flutter Web via GitHub Actions
- déploiement automatique sur GitHub Pages après chaque push sur main
